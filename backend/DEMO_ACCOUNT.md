# 데모 계정 (Demo Account)

시드 스크립트로 생성된 테스트 계정입니다.

## 로그인 정보

| 항목 | 값 |
|------|-----|
| **이메일** | `demo@anbu.app` |
| **비밀번호** | `anbu2025` |

## 시드 실행 방법

백엔드 폴더에서 다음을 실행하세요 (DB가 초기화된 상태에서).

```bash
cd Fastbuilderton/backend
python seed_demo_user.py
```

또는 `py seed_demo_user.py` (Windows)

## 생성되는 데이터

- **사용자**: 1명 (demo@anbu.app)
- **카테고리**: 가족, 직장, 친구, 연인, 멘토 (5개)
- **페르소나**: 카테고리당 2명 (총 10명)
  - 가족: 엄마(76°), 아빠(72°)
  - 직장: 팀장님(58°), 동료 지훈(58°)
  - 친구: 민수(62°), 수진(60°)
  - 연인: 지은(44°), 준호(40°)
  - 멘토: 박교수님(56°), 김선배(54°)

앱에서 **프로필** 탭 → 로그인 후 **홈**으로 돌아가면 이 계정의 카테고리/페르소나 기준으로 라다 차트가 표시됩니다.
